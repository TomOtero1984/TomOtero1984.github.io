About Me

Dynamic and creative problem solver with diverse background. Experienced in SDLC and agile development methodology from both a design and test point of view. Highly motivated self-starter proficient in a fast-paced work environment. Committed lifelong learner with background in mechanical design, robotics, and control theory now working as an embedded software test engineer. Solid understanding of data structures, algorithms, object oriented coding principles. Skilled in automated testing using Pytest, continuous integration with Buildkite, Git SVC, and Jira requirements tracking. Seeking new opportunities in the software industry.

I love writing software that interacts with the world. My first coding experiences were in robotics writing simple actuator controls for Arduino. As I learned more concepts in software and computer science, I started writing automation scripts in Python, often with interactive command line interfaces, for testing IoT and Smart Home devices. This developed my understanding of software testing principles and lead to me working as an embedded software test engineer contractor.

Highlights

  • Developed text to speech automation for smart speaker testing, simplifying the creation of new test cases.
  • Trained new employees on development and debug processes for Linux distributed system, audio verification test framework
  • Designed an RF-shielded, mesh network testbed and Python scripts for automating interoperability testing of BLE light bulbs and smart home devices
  • Simplified robotic control scripts by adopting an object-oriented design which encapsulated actuators into their own class
  • Adapted component specific Python libraries for use in general purpose Pytest application, reducing the software development effort in creating future tests

Recent Projects

  • Stewart Platform is a type of parallel manipulator that has six linear actuators attached in pairs to three position on the baseplate, crossing over to three mounting points on the top plate. The top plate is capable of six degrees of freedom. This project was used for simulating head movements to test VR headsets.
  • For Intro to Robotics at PSU, I helped create a six degree of freedom robot arm that was capable of writing the first ten letters of the alphabet in a user defined four character sequence. My role in this project was in creating the inverse kinematic path planning algorithm using Matlab and writing arduino code for controlling the actuators in the system.
  • Web Scraping project to pull metadata from archival photo information. This was an JavaScript and Node.JS exploratory project which also benefitted my wife. The ask was for a tool that could speed up the process of retrieving photo information from historical society's photo archives and placing that information into a csv. I saw this as an exceptional opportunity to explore web scraping, JavaScript, and Node.JS. Ultimately, the tool was effective, speeding my wife's productivity in her work.
